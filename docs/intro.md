# Introduction

## Git en 3 points

1. **Contrôle de version** - Git permet de suivre l'historique des modifications de votre code et de revenir à des versions antérieures si nécessaire.

2. **Collaboration** - Plusieurs développeurs peuvent travailler simultanément sur le même projet en fusionnant leurs changements de manière contrôlée.

3. **Branches** - Git permet de créer des branches isolées pour développer des fonctionnalités sans affecter la branche principale du projet.
